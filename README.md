## 📌 Project Overview
This repository contains a Jupyter Notebook designed as an educational workshop for the **Ocean Networks Canada (ONC)**. 

The project demonstrates how to use the **Oceans 3.0 API** to programmatically retrieve, clean, and visualize chlorophyll data from the Strait of Georgia. By analyzing the transition from winter baselines (0.13–0.18 µg/L) to spring bloom peaks, we explore the bloom during spring.

## 🎓 Learning Objectives
This workshop is structured to help students and hackathon participants:
1.  **Interact with APIs:** Authenticate and request real-time ocean data using the `onc` Python client.
2.  **Data Wrangling:** Convert raw API JSON responses into structured `Pandas` DataFrames.

📊 Key Data Insights
Baseline: Observed winter chlorophyll levels at ~0.15 µg/L.

Diel Cycles: Visible high-frequency "sawtooth" patterns representing daily photosynthesis cycles.
